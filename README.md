## Paradigm:  Deleted

_People tend to seek patterns and in doing so limit their thinking.  This project is about eliminating paradigms. :wave:_

**What is TINBox?**: TINBox is a collaboration platform for introducing new ideas and perspectives. 

**What is a Paradigm?**: A paradigm is a pattern, commonly referred to as a 'box' containing a defined (confined) perspective.
